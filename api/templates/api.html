GET /api/tutorials/{tags}/

Returns tutorials filtered by tags.
Multiple Tags should be separated with commas,
for e.g /api/tutorials/python,api

GET /api/tutorials/{tags}/{category}/

Returns tutorials filtered by tags and category.
Only one category can be specified at a time for e.g
/api/tutorials/python,api/video or
/api/tutorials/react/article

Available Categories :
  • article
  • video
  • docs
  • course
GET /api/tags/

Returns all the tags.
