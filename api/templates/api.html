Request Endpoint Description
GET /api/tutorials

Returns all the tutorials.
POST /api/tutorials

Submit a tutorial.
GET /api/tutorials/{tags}/

Returns tutorials filtered by tags.
Multiple Tags should be separated with commas,
for e.g /api/tutorials/python,api

GET /api/tutorials/{tags}/{type}/

Returns tutorials filtered by tags and type.
Only one type can be specified at a time for e.g
/api/tutorials/python,api/video or
/api/tutorials/react/articles

GET /api/tags/

Returns all the tags.
GET /api/latest/

Return latest 10 tutorials from tutorialdb.
